Rock Paper Scissors Python Project:

Overview:

I have developed this short, user friendly and interactable Rock Paper Scissors game with the help of a tutorial and my own skillset. This demonstrates the use of core concepts of python in an actual game.

Features:

Uses selection, functions, Validation and Error handling, aswell as pixel art as a UI, random number python library

Tools:

Python 3.12

Visual Studio Code

What I learned:

How to structure and produce an interactive game using python, how to properly validate user input and produce a robust program, using comments effectively to improve readability
